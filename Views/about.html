Introduction et description

Projet de semestre 7 à l'école d'ingénieurs CPE Lyon, promotion IRC 2021-2024.

L'objectif de notre groupe est de créer un framework WEB en PHP. Un framework ou cadre d'application Web est un cadre logiciel conçu pour prendre en charge le développement d'applications Web, découpé en plusieurs composants.

Composants d'un framework :

  • Services Web
  • Ressources Web
  • API Web

Choix de l'équipe

Développé avec PHP 7.4

Architecture logicielle :

MVC

Fonctionnalités :

  • Routeur de routes
  • Contrôleurs
  • Autoloader et namespaces
  • Moteur de templating
  • Routes annotées
  • ORM (opt.)
  • Prise en compte de fichiers d'environnement (opt.)
