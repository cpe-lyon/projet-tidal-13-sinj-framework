Introduction

SINJ Framework is a PHP Library written with PHP itself. It will help you to create a WEB server the easiest way. It uses PHP 7.4 and was written for a school projet of the french engineering school CPE Lyon.


For now, SINJ Framework does provide simple and lightweight features such as :

  • Router and routes
  • Controllers
  • Autoloader and namespaces
  • Simple templating motor
  • Simple ORM

Getting started

Routing

The routing system is made to be simple.

All your routes ake kept in a file named routes.php in the Config folder. This file does return the array of the mapped routes.


Take a look at the example :

?php

use Framework\Route;

return [
    // Books
    new Route('POST', '/book', BookController::class, 'createBook'),
    new Route('GET', '/book/delete', BookController::class, 'deleteBook'),
    new Route('GET', '/book/update', BookController::class, 'updateBook'),
    new Route('GET', '/book/all', BookController::class, 'getAll'),
    new Route('GET', '/book/find', BookController::class, 'find'),
];

The methods you can use are GET and POST. The routes need to start with a "/". The linked controllers need to be placed in the Controllers folder, and the controller's function need to be named after a real function that is in the controller otherwise it won't work.
